## Features

- **AI mood board generation from text**
- **Color palette extraction**
- **Font pairing suggestions**
- **AI-generated concept images**
- **Public gallery of mood boards**
- **Export and share**

## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Supabase (PostgreSQL, Auth, Edge Functions, Storage)
- **AI:** Claude / Gemini via API
- **Hosting:** Lovable
- **Build Tool:** Vite

## Getting Started

```bash
# Clone the repo
git clone https://github.com/LazyUnicornFounder/lazy-tones.git
cd lazy-tones

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Add your Supabase credentials to .env

# Start the dev server
npm run dev
```
